HTML TAGS!!!!!

  • title= what you see in the top bar
  • a href= a link to another website/page
  • h1-h6= a heading for the page
  • ul= the start of a list
  • li= a bullet point for the list
  • !DOCTYPE= defines the doctype
  • style= defines the style for the certain thing you would like to style
  • span= secludes a block of text(used for style)
  • strong= Defines important text
  • small= makes text smaller
  • p= makes a paragraph
  • nav= defines navigational links
  • link= commonly used to connect the HTML page to the CSS page
  • img= adds an image
  • html= defines the roots for an HTML page
  • head= defines information about the webpage
  • div= used to cinfine text/ images/ content imto a specific area
  • body= where the bulk of the information will be located
  • b= boldens the text
